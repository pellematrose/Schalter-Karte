# Schalter-Karte

Eine Schalter-Karte als Ersatz für Sicherungsautomaten im 24V-System der ELIDA.

---

## 🔧 Gedachte Verwendung

An die Schraubklemme **J1** werden **24V** und **GND** angeschlossen.  

Eine **Autosicherung** schützt die angeschlossene Last, die an **J2** angeschlossen wird.

Über den **Schiebeschalter** kann zwischen **Strompfad A und B** umgeschaltet werden:

- **Schalterstellung A**  
  → Verbraucher **A EIN**, **B AUS**

- **Schalterstellung B**  
  → Verbraucher **B EIN**, **A AUS**

Die Anschlüsse **A** und **B** an **J2** werden jeweils über den Schalter auf **24V** geschaltet.

---

## 🔄 Weitere Verwendungsmöglichkeit

Es kann auch **nur +24V** angeschlossen werden, **GND entfällt**.

In diesem Fall fungiert die Schaltung als:

> **Schaltbare Sicherung nur für den Pluspfad**

Vergleichbar mit einem klassischen **Sicherungsautomaten**.

Der angeschlossene Verbraucher erhält **GND extern**.

---

## 📏 Technische Daten

- **PCB Maße:** 17,5 mm × 80,5 mm  
- Die Abmessungen orientieren sich an einem **Sicherungsautomaten**
