# Schalter-Karte
Eine Schalter Karte als Ersatz für Sicherungsautomaten im 24V System der ELIDA.  
## Gedachte Verwendung
An die Schraubklemme J1 werden 24V und GND angeschlossen. Eine Autosicherung sichert die angeschlossene Last, die an J2 angeschlossen wird, ab. Über den Schiebeschalter kann zwischen Strompfad A und B geschaltet werden.  
Ist die Schalterstellung auf A ist der Verbraucher A an und B aus. Das gleiche gilt umgekehrt für die andere Schalterstellung.  
Die Anschlüsse A und B an J2 werden mit dem Schalter auf 24V geschaltet.  

### Weitere Verwendungsmöglichkeit
Es wird nur +24V angeschlossen und GND wird weggelassen.  
Nun fungiert die Elektronik als schaltbare Sicherung nur für den + Pfad. Vergleichbar mit einem Sicherungsautomaten. Der angeschlossene Verbraucher bekommt die GND von woanders.  
 
PCB Maße: 17,5mm x 80,5mm  
Die Maße orientieren sich an einen Sicherungsautomaten.
